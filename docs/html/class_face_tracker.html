Detailed Description

Tracks a face as it moves around. This class is a wrapper around OpenCV and provides an eassy to use interface for tracking a face between frames. The class exposes several face detection tuning parameters.

void FaceTracker::ResetTracker ( )

Stops tracking of the current face.

This Tracker will attempt to follow the same face around as it moves around the field of view of the camera. This function stops the tracking of the current face, the next time position information is requested, a new face will be selected to be tracked.
